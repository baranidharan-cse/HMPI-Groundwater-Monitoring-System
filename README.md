# HMPI Groundwater Monitoring System

AI-powered web application for monitoring groundwater heavy metal pollution using the Heavy Metal Pollution Index (HMPI).

## 📌 Problem Statement
Develop an automated system that calculates HMPI from groundwater heavy metal concentration data and visualizes pollution levels.

## 🚀 Features

- Automatic HMPI calculation
- Groundwater dataset upload
- Water quality classification
- Interactive pollution map
- Dashboard analytics
- AI-based pollution prediction

## 📊 HMPI Formula

HMPI = Σ(Wi × Qi) / Σ(Wi)

Where:

Qi = (Mi − Ii) / (Si − Ii) × 100
